# Interpro Blues — Lineups (Web)

Web/PWA version of the Interpro Blues lineup tool.

## Stack
- Frontend: static HTML/CSS/vanilla JS
- Backend: Supabase (Postgres + auth + RLS)
- Hosting: Vercel

## Local dev
Just open `index.html` in a browser — no build step.

## Deploy
Pushed to `main` on GitHub → auto-deployed by Vercel.
